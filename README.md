# Silver Price Calculator & Sales Dashboard

This Streamlit app demonstrates a Silver Price Calculator and a Silver Sales Dashboard for India.

How to run locally:

1. Create a virtual environment and install requirements:

```bash
python -m venv .venv
\.venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the app:

```bash
streamlit run streamlit_app.py
```

Notes:
- Sample data is included in the `data/` folder. Replace these files with your provided datasets for full analysis.
- For the map, `geopandas` is required. If installation issues occur on Windows, consult GeoPandas installation docs.
